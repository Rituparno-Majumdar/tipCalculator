Tip Calculator
Welcome to the Tip Calculator! This is a simple Python program that helps you calculate the amount each person should pay when splitting a bill, including the tip.

Getting Started
To use the Tip Calculator, follow these steps:

1. Run the program.
2. When prompted, enter the total bill amount in rupees.
3. When prompted, enter the percentage of tip you would like to give (10, 12, or 15).
4. When prompted, enter the number of people involved in splitting the bill.
5. The program will calculate the total tip based on the tip percentage and the total bill amount.
6. It will then calculate the amount each person should pay by dividing the tip equally among the number of people.
7. The program will display the result, showing how much each person should pay in rupees.

Prerequisites
Python 3.x

Usage

1. Open a terminal or command prompt.
2. Navigate to the directory where the code file is located.
3. Run the following command to execute the program: python tipCalculator.py
